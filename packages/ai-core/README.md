# @ais-chat/ai-core

A package for communicating with AI providers and LLMs.

## Features

- Chat completions
- Embeddings
- Image generation
- API key management including billing and access control
- Zod schema validation for provider responses

## Installation

This package is part of the AIS.chat monorepo and is not published separately.

## Scripts

- `pnpm test` - Run tests
- `pnpm test:watch` - Run tests in watch mode
- `pnpm test:coverage` - Run tests with coverage
- `pnpm lint` - Lint the code
- `pnpm types` - Check TypeScript types
